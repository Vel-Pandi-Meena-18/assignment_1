## Domain Introduction
The cryptocurrency domain deals with digital assets that are traded online using blockchain technology. It focuses on analyzing market trends, prices, volumes, and investor behavior. This domain helps in understanding financial risks and opportunities.

---

## Project Introduction
This project analyzes cryptocurrency market data using Python, MySQL, and Streamlit. It provides insights through data visualization and interactive dashboards.

---

## Objective of the Project
The main objective is to collect, clean, store, and analyze crypto market data. It aims to help users understand price trends and market performance.

---

## ELT Approach
Extract data from API and MongoDB, Load it into MySQL database, and Transform it using SQL and Python for analysis and visualization.

---

## Data Migration
The data was transferred from MongoDB to MySQL using Python scripts. This helped in structured storage and easy querying using SQL.

---

## Exploratory Data Analysis (EDA)
EDA showed that Bitcoin and Ethereum dominate market capitalization. Some cryptocurrencies showed high volatility and irregular trading volumes.

---

## Feature Engineering
Date conversion, currency conversion to INR, handling missing values, and normalization were performed to improve data quality.

---

## Statistical Technique
Correlation analysis was used to understand the relationship between market cap and price, as it helps identify dependent variables.

---

## Conclusion
The project revealed major trends in crypto markets and identified top-performing cryptocurrencies. It helped in understanding market behavior using data analysis.

---

## Business Suggestion
Investors should focus on high-market-cap and stable cryptocurrencies. Monitoring volatility can reduce financial risks.
